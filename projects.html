Academic Projects

JavaFX Appointment Scheduling System

This application incorporates JavaFX and Java to create a desktop graphical user interface used to simulate an appointment-scheduling system. MySQL functionality allows for persistent storage of the data a potential user would enter. This program required less than two thousands lines of code contained within several packages, organized in the Model-View-Controller pattern.

Scheduler Mobile App

Designed to run on Android mobile phones created in Android Studio, this program consists of several screens. They show cumulatively the individual terms, assessments, and courses for a simulated student, saved on a locally-available Room database. Each element of the app contains fields for text input and lists which update frequently, requiring the implementation of error-handling code.

Inventory Management System

Established using JavaFX and Java, this program represents an inventory management system without persistent storage. This project provided an important lesson in Object-Oriented Programming, requiring the proper utilization of inheritance. In addition, each screen in the graphical user interface embodied either parts or products which intricately meshed together in different ways, needing the input of error-handling controls as well.

Student Roster

A challenging assignment needing the creation of a student roster program coded entirely in C++, this program involved an introduction to Object- Oriented Programming principles, outputting a screen based on an array of strings representing student information. This application needed the exact reproduction of an example screen, necessitating the proper creation of a sound, interconnected data structure.

Personal Projects

Text-Based Sports Trivia Quiz

This project involves a simple application programmed entirely in Python producing a sports trivia quiz with pre-set questions and answers. This program uses persistent storage of individual and overall high scores, implemented with a SQLite database file. Two Python files exist: one containg the code for running the quiz and another for establishing SQL queries and connectivity.

Grocery List Mobile App

Produced for personal use using Android Studio, this mobile app allows for the adding and deleting of items in a simple interface using one screen. Within the screen, the user can enter data in a single text field that dynamically updates the list. To delete data, one can simply swipe on the item to the right of the screen. A Room database keeps data present in case the user shuts down the app and reboots later.
