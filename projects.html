JavaFX Appointment Scheduling System

This application incorporates JavaFX and Java to create a desktop GUI used to simulate an appointment-scheduling system. MySQL functionality allows for persistent storage of the data a potential user would enter. This program required less than two thousands lines of code contained within several packages.

Scheduler Mobile App

Designed to run on Android mobile phones using Android Studio, this program consists of several screens. They show cumulatively the individual terms, assessments, and courses for a simulated student, saved on a locally-available Room database. Each element of the app contains fields for text input and lists which update frequently, requiring the implementation of error-handling code.
