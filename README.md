This repository contains the files for my personal Github Page. Three HTML and one
CSS file make up the website. The website's size did not require more than one
CSS file.

The index.html file creates the home page. The about.html file makes an About Me
page with a brief synopsis of who I am. The projects.html file allows the
Projects page to appear, showing the various academic and personal projects I
implemented.

The repository contains several other files: a photo of myself, a favicon icon
file, and a simple background image.

Go to https://gabholli.github.io to visit the website!
