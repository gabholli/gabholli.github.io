About Me

I'm a full-stack web developer based in West Sacramento, CA, with a focus on building real-world applications using React, Next.js, TypeScript, and Supabase. My recent projects include a full-stack dog breed database and a recipe discovery app — both featuring user authentication, PostgreSQL-backed data persistence, and responsive design built for production.

I currently work at MindHome, a home robotics company, where I develop and maintain web experiences using Shopify and modern front-end tools. I hold a Bachelor of Science in Software Development from Western Governors University.

I'm always open to new opportunities — feel free to reach out on LinkedIn!
